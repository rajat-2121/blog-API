### How to run
1. Clone the repository to your local machine using git bash.
<pre><code>git clone 
</code></pre>

2. Open the folder in VS Code and install node_module
<pre><code>npm i
</code></pre>

3. Run the test cases using following command
<pre><code>npm test
</code></pre>

( Your system should have node.js and mongoDB )
